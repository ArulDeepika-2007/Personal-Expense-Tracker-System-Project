# Personal Expense Tracker System

## Project Description
The Personal Expense Tracker System is a web application that helps users manage their daily income and expenses. Users can add, view, update, and delete transactions, track spending habits, and monitor their financial balance.

## Technology Stack
Frontend: HTML, CSS
Backend: Java, Spring Boot
Database: MySQL

## Features
- Add Income
- Add Expenses
- View Transactions
- Update Records
- Delete Records
- Check Balance
- Generate Reports

## Objective
To help users track and manage their personal finances efficiently by maintaining organized records of income and expenses.
